⚛️ Chat App (React + WebSockets)

A real-time chat application built using React and WebSockets, featuring dynamic UI updates and component-based architecture.

EDIT : I have also hosted this Project on Vercel ( Frontend) and Render ( Backend ). 

URL : https://chat-app-using-react-ebon.vercel.app

---

📌 About

This project demonstrates how modern frontend development with React can be combined with WebSockets to create fast and interactive real-time applications.


---

🚀 Features

💬 Real-time messaging

👥 Multi-user chat

⚡ Instant UI updates

🔄 Persistent WebSocket connection

🧩 Component-based UI



---

🧠 Concepts Covered

React components

Hooks (useState, useEffect)

WebSocket integration

State management

Real-time UI updates

⚙️ Getting Started

1. Clone the repository:

git clone https://github.com/devKartikeya/chat-app-react.git


2. Install dependencies:

npm install


3. Start the app:

npm start
